===
tem
===

SYNOPSIS
========

.. code-block:: none

   tem [--version] [--help]
   tem [<options>] <command>

DESCRIPTION
===========

tem is a lightweight yet feature-packed template/snippet manager for the
terminal.

OPTIONS
=======

`-v`\ , `--version`
   Prints the currently installed version of tem.
`-h`\ , `--help`
   Prints the synopsis and the list of commands.
